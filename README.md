# 📝 HWPX to Markdown Converter (Streamlit Web App)

한글(HWPX) 문서를 깔끔한 구조의 **Markdown(.md) 파일로 완벽하게 변환**해 주는 Streamlit 기반 웹 애플리케이션입니다.

기존 변환기들의 고질적인 문제였던 **표(Table) 안의 표(중첩 표) 구조 붕괴 문제**를 해결하고, 여러 파일을 한 번에 변환하여 이미지와 함께 ZIP으로 다운로드할 수 있는 강력한 기능을 제공합니다.

## ✨ 주요 기능 (Features)

* 📄 **정확한 서식 변환**: 제목(Heading), **굵게**, *기울임*, ~~취소선~~, <sup>위첨자</sup>, <sub>아래첨자</sub> 등 한글 문서의 핵심 서식을 마크다운 문법 및 HTML 태그로 완벽히 치환합니다.
* 📊 **강력한 표(Table) 파싱 엔진**: 
  * 텍스트와 섞여 있는 표 구조를 안전하게 분리합니다.
  * 마크다운 표 안에 줄바꿈(`<br>`)을 지원하여 표 붕괴를 방지합니다.
  * **표 안의 표(Nested Table)**를 HTML `<table>` 태그로 변환하여 행합치기/열합치기 구조를 온전히 보존합니다.
* 🛡️ **데이터 분석 최적화 (LLM 친화적)**:
  * 마크다운 최상단에 메타데이터(Frontmatter)를 자동 삽입하여 문서 출처와 변환 일시를 기록합니다.
  * 본문의 물결표(`~`)를 이스케이프(`\~`) 처리하여 마크다운 취소선 문법과 충돌하지 않도록 방어합니다.
* 📦 **다중 파일 일괄 변환 & ZIP 다운로드**: 여러 개의 `.hwpx` 파일을 한 번에 업로드하고, 변환된 `.md` 파일들과 추출된 이미지들을 구조화된 폴더(ZIP) 형태로 일괄 다운로드할 수 있습니다.
* 🖼️ **이미지 자동 추출**: 문서 내 삽입된 이미지(.png, .jpg 등)를 원본 화질 그대로 추출하여 마크다운과 함께 제공합니다.
